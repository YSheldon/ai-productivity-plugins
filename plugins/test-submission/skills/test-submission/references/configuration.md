# Configuration

- `mail_account.profile` and `mail_account.email` come from `imap-smtp-mail`; never store passwords here.
- `submission_gate_address` is the gate mailbox recipient for `【提测】` mail.
- `mandatory_checks_by_module` defines the strong checks that cannot be disabled.
- `svn_mandatory_checks` defines the fixed-revision provenance and protected-CI retrieval checks; it intentionally does not require sender-side file hashes, signatures, or cloud-scan results.
- `default_optional_checks` may be empty, but effective checks may never be empty.
+- `product_gate_preview_config` points to the local preview config used by the locked `product-release-gate` CLI. It is required for local artifacts, not for SVN-only submission readiness.
- `dependency_lock` and `dependency_lock_sha256` must be generated by setup and must not be hand-edited.
